# nullius

*Nullius in verba* — "take nobody's word for it."

A falsify-first toolkit for backtests: lookahead detection with negative controls, skill
placebos, honest forecast evaluation, and multiple-testing deflation (Deflated Sharpe, PBO).

**Status:** this is a `0.0.0` placeholder reserving the name on PyPI. The library is in
active development — watch this space.
