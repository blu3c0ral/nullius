<h1 align="center">Nullius</h1>

<p align="center">
  <em>nullius in verba</em> — take nobody's word for it
</p>

<p align="center">
  A falsify-first toolkit for backtests: lookahead detection with negative controls,<br>
  skill placebos, honest forecast evaluation, and multiple-testing deflation.
</p>

---

Every backtesting library helps you **produce** a performance number. `nullius` helps you
**attack** one — the missing "falsify-first" layer for quants who'd rather kill a false
result early than discover it in production.

> **Status:** this is a `0.0.0` placeholder reserving the name on PyPI. The library is in
> active development — watch this space.
